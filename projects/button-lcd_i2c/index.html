    • Button Flag with LiquidCrystal I2C

    This project demonstrates how to display a message on an LCD screen using the LiquidCrystal I2C library when a button is pressed.

    Components Required

    • Arduino Board
    • 16x2 LCD with I2C Module
    • Push Button
    • Resistors
    • Jumper Wires
    • Breadboard

    Code

    Below is the Arduino code for this project:

    
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);

#define button_pin 12
#define led_pin 13

void setup() {
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  pinMode(button_pin, INPUT_PULLUP);
  //pinMode(led_pin, OUTPUT);
}

void loop() {
  //digitalWrite(led_pin, !digitalRead(button_pin));
  lcd.clear();
  if (!digitalRead(button_pin)) {
    lcd.print("Button pressed");
  } else {
    lcd.print("Button relessed");
  }
  delay(100);
}

    Instructions

    1. Connect the components as per the circuit diagram.
    2. Upload the provided code to your Arduino board.
    3. Press the button to see the message on the LCD screen.
